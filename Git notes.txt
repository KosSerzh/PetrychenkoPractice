git status

git add -A добавляет все к индексированию
git add '.\folder1\folder2\file' добавл папку или файл к индексированию

git commit -a -m'' добавляет все в локал
git commit '.\folder1\folder2\file' -m'' добавл папку или файл в локал

git push добавляет все из commit в удаленный репозиторий
git push '.\folder1\folder2\file' добавл папку или файл в удаленный репозиторий

git checkout возвр в main ветку
git checkout name_of_branch переключ в выбранную ветку

git branch все доступные ветки; Выделяет ветку в которой находишься

fatal: The current branch name_of_branch has no upstream branch.
git push --set-upstream origin name_of_branch добавляет локальн ветку в удаленный репо
